## all()

```typescript
all(ticker: string): Promise<ApiResponse>
```

Get all sentiment data for a ticker.

<ParamField path="ticker" type="string" required>
  Stock ticker symbol
</ParamField>

### Example

```typescript
const sentiment = await client.sentiment.all('AAPL');
```
